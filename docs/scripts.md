## Scripts

### `test` – run the tests

```sh
npm run test
```

### `coverage` – generate test coverage

```sh
npm run coverage
```

### `view-coverage` – view test coverage

```sh
npm run view-coverage
```

### `lint` – lint the codebase

```sh
npm run lint
```

### `write-readme` – generate the README

```sh
npm run write-readme
```
